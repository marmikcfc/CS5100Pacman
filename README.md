# CS5100Pacman
Implementing AI algorithms on MS Pacman VS Ghost Framework for CS 5100 - Foundations of Artificial Intelligence 

## Algorithms Implemented ##

* Uninformed, Blind Search
 * Breadth-First Search 
 * Depth-First Search
 * Iterative Deepening

* Informed Search
 * Hill Climbing

* Adversarial Search 
 * Alpha Beta
 
* Evolutionary Algorithms 
 * Genetic Algorithm

* Supervised Learning 
 * Decision Tree
 * K-Nearest Neighbour
