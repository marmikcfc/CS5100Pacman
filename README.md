# CS5100Pacman
Implementing AI algorithms on MS Pacman VS Ghost Framework for CS 5100 - Foundations of Artificial Intelligence 

## Algorithms Implemented ##

* Uninformed, Blind Search
 * Breadth-First Search 
 * Depth-First Search
 * Iterative Deepening

* Informed Search
 * Simulated Annealing
 * Hill Climbing
